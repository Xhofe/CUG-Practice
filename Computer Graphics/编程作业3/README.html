二、解析参数与捕获t

2.1 解析参数

第一个是一个字符（e,t,r),选择求解器（Euler法，梯形法或RK4）。第二个是可选的求解器步长。在initSystem 中实现，使用不同的方法即new不同TimeStepper的子类即可，对于求解器步长，在stepSystem中将h全局化，然后再初始化。

2.2 捕获t

在keyboardFunc中添加对t的捕获，切换不同的系统与求解器类似，new不同ParticleSystem的子类即可。
